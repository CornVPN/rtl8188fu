# Установка драйвера RTL8188FU на Bazzite OS (Fedora Silverblue)

Инструкция по ручной сборке и настройке автозагрузки Wi-Fi адаптера Realtek RTL8188FU (USB) в неизменяемых (immutable) Linux-системах, где стандартный DKMS не поддерживается.

## 🛠️ Шаг 1. Подготовка системы и зависимостей

Убедитесь, что в вашей системе установлены заголовки ядра (kernel-devel) через наложение слоев:
```bash
sudo rpm-ostree install kernel-devel
```
*Если пакет уже установлен, шаг можно пропустить. Если он установился только что — обязательно перезагрузите ПК.*

## 🔨 Шаг 2. Сборка модуля

1. Склонируйте исходники драйвера (например, популярный репозиторий от `kelebek33` или `KubaOman`) и перейдите в папку:
```bash
cd ~/rtl8188fu
```
2. Очистите кэш и скомпилируйте модуль ядра вручную:
```bash
make clean
make
```

## 🚫 Шаг 3. Блокировка конфликтующего драйвера

Встроенный в ядро модуль `rtl8xxxu` конфликтует с этим чипом. Навсегда заблокируем его через параметры загрузчика:
```bash
sudo rpm-ostree kargs --append="modprobe.blacklist=rtl8xxxu"
```
**Обязательно перезагрузите компьютер**, чтобы применить изменения параметров ядра!

## ⚙️ Шаг 4. Настройка стабильности NetworkManager

Чтобы избежать бесконечного сканирования и сбросов соединения, отключите случайные MAC-адреса и управление питанием:

```bash
sudo nano /etc/NetworkManager/conf.d/99-rtl-fix.conf
```
Вставьте следующие строки:
```ini
[device]
wifi.scan-rand-mac-address=no

[connection]
wifi.cloned-mac-address=preserve
wifi.powersave=2
```

## 🚀 Шаг 5. Настройка автозагрузки (Решение для Bazzite)

Так как SELinux в Bazzite блокирует стандартные службы `systemd`, загружающие модули из домашних папок, используем системный скрипт `rc.local`:

1. Откройте или создайте файл автозагрузки:
```bash
sudo nano /etc/rc.d/rc.local
```
2. Добавьте в него конфигурацию с флагами **отключения агрессивного энергосбережения** (без них соединение будет обрываться) и фиксацией на **20 МГц** для максимальной стабильности:
```bash
#!/bin/bash
/usr/sbin/modprobe cfg80211
/usr/sbin/insmod /home/icuxu/rtl8188fu/rtl8188fu.ko rtw_ht_enable=0 rtw_ips_mode=0 rtw_power_mgnt=0
exit 0
```
3. Сделайте скрипт исполняемым:
```bash
sudo chmod +x /etc/rc.d/rc.local
```

Запустите его вручную для проверки (`sudo /etc/rc.d/rc.local`) или просто перезагрузите систему. Wi-Fi будет подниматься автоматически при каждом включении ПК!
